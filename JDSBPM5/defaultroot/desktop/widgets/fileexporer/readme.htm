-JSP在线文件管理系统 v1.0beta Build 030110
-----------------------------------------------

一、功能简介(v1.0b):
-----------------------------------------------
1、对于服务器上的文件、目录进行批量的移动、复制、删除。
2、无须Bean的多文件同时上传。
3、文件在线编辑。

二、安装说明(v1.0b):
-----------------------------------------------
1、测试环境：
   Server: Win2000+J2SDK1.4.1+Resin2.1.6
   lient: IE 6.0.2600.0000
2、无须虚拟目录支持，放入发布目录下即可。
3、index.jsp为登录页面（现在还没加入权限限制）
